Overview

TI - Python adapter is a solution that allows teachers and students to program in Python on the TI-83 Premium CE calculator. This project initially started as a development of an external plugin accessory for the company’s best seller product.

Product Goal Analysis

As a completely new feature for this product, it was very important to understand the following 3 parts of the product: the user needs, the company’s business goals, and the product goals with their constraints. The process for this project started with putting together these 3 parts:

Design problem

“ Transform the current Note editor into a Python script editor. “

Fortunately, we didn't have to start this product from nothing; the calculator already had a feature called ”Note editor” that we could be used as a starting point.

User research

Looking for areas of opportunities to improve the user experience the team decided to create a high-fidelity mockup typing a short python script in the already existing “Note editor” feature. The purpose of this mockup was to get some feedback from python developers and perform a heuristic evaluation. Here are the top 5 user pain points that were identified during the user research.

  1. The indentation is not clear.
  2. Lower case letters with descenders like “p” and “y”look uppercased.
  3. The overflow wrapped words are hard to read.
  4. Scrolling down through all the code is cumbersome.
  5. The code is hard to understand when the text is all in one color.

Since graphing calculators have a custom screen size; we looked for code editor apps for smartphones to analyze how they treat code strings on small-size screens. This gave us the idea of how to solve some of the user pain points and led us to the final design proposal.
